--- title: Auf geht's ins JavaLand... tags: - vortrag - javascript - avatar - javaland - java - konferenz - nodejs - oracle --- Morgen ist es soweit: das JavaLand öffnet zum ersten Mal seine Türen und lässt Wissenshungrige rund um das Thema Java (als Sprache, als Laufzeitumgebung, ...) einreisen. Ich freue mich schon sehr auf diese erste Community-Konferenz in Deutschland. Könnte fast sowas wie eine deutsche (europäische?) JavaOne werden: jede Menge interessante Vorträge und Speaker!
Zusammen mit Heiko werde ich am Mittwoch um 16:00 Uhr im Schauspielhaus über das Project Avatar reden - mehr als nur Node.js auf der JVM, quasi Enterprise Java, programmiert in JavaScript. Wird sicherlich interessant und spannend!
Am Dienstag um 15:45 Uhr interviewt mich Stephen Chin von Oracle im Rahmen seiner NightHackings noch zum gleichen Thema. Das Interview soll live ins Internet übertragen werden und auf www.NightHacking.com zu sehen sein. Am Donnerstag kommt Steve übrigens dann zur JUG Darmstadt und stellt Java 8 live im Einsatz mit IoT Devices wie Raspberry Pi und Lego Mindstorms vor. Ebenfalls live gestreamt!
Ich such' dann mal meine Ausweispapiere und mein Visum fürs JavaLand zusammen. Lt. schon angereister Personen, gibt es aber keine Probleme bei der Einreise. ;-)
